Help & Guides

Quick references for preparing clean print requests and estimating accurate pricing.

How to find model weight

  • Open the model page on MakerWorld or Printables.
  • Copy the specs area that includes weight and estimated print time.
  • Use the Auto-Fill Assist on the Order page to populate fields quickly.

Choosing filament colors

  • Visit the Materials tab to preview available colors.
  • If your first choice is unavailable, select the nearest backup color.
  • For multi-color jobs, mention extra details in your project name.

Order lifecycle

Orders move through: Pending Quote, Approved, Printing, Quality Check, then Delivered. You can monitor progress from Home and My History.

Need support?

Open any order and use the built-in message thread for revisions, clarifications, or timeline updates.
